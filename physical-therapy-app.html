Physical Therapy App

An application to simplify physical therapy routines

This project was inspired by a recent knee injury and a need to simplify my physical therapy routine. I had a lot of fun making this, and it was very rewarding to see it come together as something I will actually use. It was built with vanilla JavaScript, Materialize CSS, SASS, local Storage, and uses the Wger.de REST API. Users can add their own custom exercises or search and pull in new ones from the API. The stopwatch is tied to each exercise which allows modification of other exercises without reseting the current timer. The code was all written by myself with a little help on the stopwatch math calculations from Wes Bos' JavaScript challenges.
